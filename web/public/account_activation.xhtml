Neden? GSM Aktivasyon'u

Web sitemize üye olabilmeniz için gsm numarası gereklidir. Üyelik işlemleri sırasında vermiş olduğunuz gsm bilgilerinin size ait olduğunu doğrulamak için bu aktivasyon işlemlerini yapıyoruz. Olurda ilerleyen zamanlarda web sitemize giriş bilgilerinizi unursanız size yeniden giriş bilgileri hatırlatmamız için sisteme girmiş olduğunuz gsm bilgileri kullanılacaktır. Size özel oluşturulan üyeliğin başkalarının eline geçmemesi için sadece kendinize ait olan gsm numarası bilgileriyle üye olunuz.
